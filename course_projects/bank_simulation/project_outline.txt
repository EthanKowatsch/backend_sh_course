### 1. File Structure

```
mini-bank/
├─ src/
│   ├─ index.js               # Main entry point & console loop
│   ├─ navigation.js          # Handles CLI menu & user input navigation
│   ├─ models/
│   │   ├─ Account.js         # Account class with createAccount static method
│   │   ├─ Transaction.js     # Transaction class
│   │   └─ errors.js          # Custom error classes
│   ├─ utils/
│   │   └─ jsonHandler.js     # Read/write JSON helper functions
├─ data/
│   ├─ accounts.json          # Stores all accounts
│   └─ transactions.json      # Stores all transactions
```

### 2. Class & Inheritance Design

**Account Class**

```
Properties:
- accountNumber (auto-generated)
- userName (string)
- balance (number, default 0)
- accountType (string: savings, investment, main, etc.)

Methods:
- deposit(amount)
- withdraw(amount)
- transfer(amount, targetAccountNumber)  # logs two transactions: transfer out/in
- getAccountInfo() -> returns account summary

Static Methods:
- createAccount(userName, accountType, accountsArray) -> creates new Account and adds to accountsArray
- generateAccountNumber()
- printAccounts()
- printSpecificAccount(accountNumber)
```

**Transaction Class**

```
Properties:
- type (deposit, withdraw, transfer out, transfer in)
- amount
- accountNumberAffected
- timestamp

Methods:
- logTransaction() -> writes transaction to transactions.json
```

**Custom Error Classes (errors.js)**

```
- InvalidAmountError
- InsufficientFundsError
- AccountNotFoundError
- InvalidTransactionTypeError
```

### 5. Utilities / Static Methods

- `generateAccountNumber()` → auto increments account numbers for new accounts
- `findAccount(accountNumber, accountsArray)` → static helper to locate account in JSON
- `readJSON(filePath)` / `writeJSON(filePath, data)` → helpers for JSON storage

### 6. Navigation File (CLI Loop)

- Loops the menu until “Exit” is selected
- Calls the relevant class methods for create/deposit/withdraw/transfer
- Handles exceptions with try/catch and prints custom error messages

### 7. Development Flow

1. Build `Account.js`, `Transaction.js`, and `errors.js` first
2. Implement JSON read/write helpers (`jsonHandler.js`)
3. Build main menu loop in `navigation.js`
4. Test each transaction type individually
5. Add proper error handling
6. Integrate `Transaction` logging for every operation
